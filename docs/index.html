Installing

Use npm i hanime.tv.

Example

Downloading a video

const { download } = require("hanime.tv")

download("url")
    .then(console.log)

Performing a search.

const { search } = require("hanime.tv")

search("itadaki")
    .then(v => v.videos)

Getting a specific video info.

const { getVideoInfo } = require("hanime.tv")

getVideoInfo(1315)
    .then(console.log)
